Sure bhai! Let's break down all the commonly used components of the `Math` object in **JavaScript**. This is a **built-in object** that provides properties and methods for mathematical constants and functions.

I'll explain them in a grouped and simple way, with **examples** so you can remember easily.

---

### ‚úÖ **1. Basic Utility Functions**

These are commonly used to find max/min, absolute value, etc.

| Function              | Description                                            | Example                   |
| --------------------- | ------------------------------------------------------ | ------------------------- |
| `Math.max(a, b, ...)` | Returns the **largest** number                         | `Math.max(3, 5, 1)` ‚Üí `5` |
| `Math.min(a, b, ...)` | Returns the **smallest** number                        | `Math.min(3, 5, 1)` ‚Üí `1` |
| `Math.abs(x)`         | Returns the **absolute** value (no negative)           | `Math.abs(-4)` ‚Üí `4`      |
| `Math.sign(x)`        | Returns `1` if positive, `-1` if negative, `0` if zero | `Math.sign(-3)` ‚Üí `-1`    |

---

### ‚úÖ **2. Rounding Functions**

Used to round numbers in different ways.

| Function        | Description                        | Example                                          |
| --------------- | ---------------------------------- | ------------------------------------------------ |
| `Math.round(x)` | Rounds to the nearest integer      | `Math.round(4.6)` ‚Üí `5`, `Math.round(4.4)` ‚Üí `4` |
| `Math.floor(x)` | Rounds **down** to nearest integer | `Math.floor(4.9)` ‚Üí `4`                          |
| `Math.ceil(x)`  | Rounds **up** to nearest integer   | `Math.ceil(4.1)` ‚Üí `5`                           |
| `Math.trunc(x)` | Removes decimal part               | `Math.trunc(4.7)` ‚Üí `4`                          |

---

### ‚úÖ **3. Power and Roots**

For exponentiation and square roots.

| Function              | Description                  | Example                |
| --------------------- | ---------------------------- | ---------------------- |
| `Math.pow(base, exp)` | `base` raised to power `exp` | `Math.pow(2, 3)` ‚Üí `8` |
| `Math.sqrt(x)`        | Square root                  | `Math.sqrt(16)` ‚Üí `4`  |
| `Math.cbrt(x)`        | Cube root                    | `Math.cbrt(27)` ‚Üí `3`  |

üëâ ES6 introduced `**` for power: `2 ** 3` ‚Üí `8`

---

### ‚úÖ **4. Random & Constants**

Useful in games, probability, etc.

| Function / Constant | Description                               | Example                       |
| ------------------- | ----------------------------------------- | ----------------------------- |
| `Math.random()`     | Returns random number between `0` and `1` | `Math.random()` ‚Üí `0.3456...` |
| `Math.PI`           | Value of œÄ (Pi)                           | `Math.PI` ‚Üí `3.14159...`      |
| `Math.E`            | Euler's number (‚âà 2.718)                  | `Math.E` ‚Üí `2.718...`         |

üëâ Random in a range:

```js
// Random number between 1 and 100
Math.floor(Math.random() * 100) + 1;
```

---

### ‚úÖ **5. Trigonometric Functions**

Used in geometry, graphics, etc.

| Function      | Description                | Example                       |
| ------------- | -------------------------- | ----------------------------- |
| `Math.sin(x)` | Sine of angle (in radians) | `Math.sin(Math.PI / 2)` ‚Üí `1` |
| `Math.cos(x)` | Cosine                     | `Math.cos(0)` ‚Üí `1`           |
| `Math.tan(x)` | Tangent                    | `Math.tan(0)` ‚Üí `0`           |

üëâ All angles must be in **radians**.

### ‚úÖ **6. Logarithmic Functions**

| `Math.log2(x)`  | Base-2 log           | `Math.log2(8)` ‚Üí `3`     |
